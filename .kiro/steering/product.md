# Product: Kiro Rider Light

A light color theme for VS Code and Kiro IDE, inspired by JetBrains Rider Light.

## Goals

- Consistent syntax colors across all supported languages (Kotlin, Java, TypeScript, JavaScript, Python, Markdown, YAML, JSON, HTML, CSS) -- a class is always purple, a function always green, regardless of language
- WCAG AA contrast (4.5:1 minimum) for all syntax colors on white
- Dedicated semantic highlighting scopes for Kotlin LSP and basedpyright
- 465 UI colors covering editor, terminal, debug, testing, VCS, and more
- Full 16-color ANSI terminal palette at perceptually uniform lightness

## Syntax Palette (fixed roles)

| Role                   | Color   |
|------------------------|---------|
| Functions              | Green   |
| Types / Classes        | Purple  |
| Keywords               | Blue    |
| Fields / Properties    | Teal    |
| Strings                | Brown   |
| Numbers                | Magenta |
| Comments               | Muted green |
| Metadata / Annotations | Olive   |

## Distribution

Published to VS Code Marketplace and Open VSX as `NikolaiFedorov.kiro-rider-light`.
